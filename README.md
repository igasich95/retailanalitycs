# Excel Table

Адаптивный веб-каркас для загрузки Excel-файлов и будущего отображения данных в таблице.

Для просмотра без установки и сервера просто откройте `index.html` двойным кликом. После выбора файла нажмите кнопку подтверждения: приложение прочитает только вкладку `Retail` и покажет шесть настроенных столбцов.

## Запуск

```bash
npm install
npm run dev
```

Сборка для публикации:

```bash
npm run build
```

## Структура

- `src/app` — сборка приложения и страницы;
- `src/features` — независимые пользовательские сценарии, сейчас только выбор Excel-файла;
- `src/shared` — общие компоненты;
- `src/styles` — глобальные стили и дизайн-токены.
